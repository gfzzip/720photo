## 🚀 如何使用？

我们提供两种使用方式，请根据您的需求选择：

### **1. 线上部署 (推荐)**

这是最终的部署方式。操作非常简单：

*   只需将本压缩包内的**所有文件和文件夹**，完整地上传到您的网站服务器或任何静态文件托管平台（如 Vercel, Netlify, 或对象存储服务）。
*   之后，通过访问您服务器上的 `index.html` 文件，即可公开访问您的全景漫游项目。

### **2. 本地快速预览**

在上传到服务器之前，我们推荐您先在本地进行预览，以确保所有内容都符合预期。

现代浏览器出于安全限制，直接双击打开 `index.html` 文件，可能会导致热点、视频等部分高级功能无法正常工作。为此，我们为您内置了一个迷你网页服务器。

您只需根据您的操作系统，运行对应的文件，即可在本地完美预览您的项目：

#### **🖥️ Windows 用户**

1.  直接**双击**运行 `windows点击运行.exe` 文件。
2.  程序会自动在您的默认浏览器中打开预览页面。

**注意：** 首次运行时，Windows 防火墙或杀毒软件可能会弹出安全提示，请选择 **“允许访问”** 或 **“信任”**。这是正常现象，请放心使用。

#### ** macOS 用户**

1.  直接**双击**运行 `mac点击运行` 文件。
2.  程序会自动在您的默认浏览器中打开预览页面。

**遇到问题？** 如果提示“无法打开，因为来自未识别的开发者”，请按以下步骤操作（只需在首次运行时操作一次）：
1.  **右键点击** `mac点击运行` 文件，然后选择 **“打开”**。
2.  在弹出的新对话框中，再次点击 **“打开”** 即可。

#### **🐧 Linux 用户**

1.  在当前项目文件夹中打开您的终端 (Terminal)。
2.  首先，给文件添加执行权限（只需操作一次）：
    ```bash
    chmod +x linux命令行运行
    ```
3.  然后，运行该文件：
    ```bash
    ./linux命令行运行
    ```
4.  终端会显示一个本地网址（如 `http://127.0.0.1:8080`），请将此网址复制到浏览器中打开。
